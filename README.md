
# Chat-WebSocket

Este é um chat aonde e possivel interagir com outros usuarios em um documento de conversa, cada documento tem seu assunto especifico.


## Instalação

Instale Chat com npm

```bash
  cd Chat-WebSocket
  npm install
  npm run dev
```
    
## Funcionalidades

- Chat em tempo real
- CRUD de documentos
- Login e cadastro
